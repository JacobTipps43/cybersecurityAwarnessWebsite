Check sender and links. Before clicking any links or opening attachments check the sender address, look for typos, verify unexpected requests by calling or messaging the sender via known channels.

Keep software updated. Install updates for OS, browsers and critical tools promptly — updates often patch security issues attackers exploit.

Use secure networks. Avoid public Wi-Fi for sensitive tasks; use a trusted VPN and prefer known cellular or home networks for confidential work.

Enable multi-factor authentication (MFA). MFA greatly reduces account takeover risk — enable whenever available (especially for email, cloud storage, admin tools).

If you suspect compromise. Disconnect from the network, change passwords from a known-good device, notify IT/security, and follow incident reporting procedures.
